# 🧠 AI PC Controller

A Windows-based AI automation system that connects large language models (LLMs) to real system execution.

It allows AI to operate the local computer through structured tool interfaces, turning it into a practical desktop agent capable of executing real tasks.

---

## ⚙️ Features

- 📁 File system operations (read / write / move / delete)
- 💻 CMD command execution
- 🖱️ Mouse and UI automation
- 🪟 Windows API control (windows, processes, messages)
- 🌐 HTTP requests
- 🧠 Dynamic C# script execution
- 🔁 Multi-step execution with memory tracking

---

## ⚠️ Important Warning

This system allows AI to execute **CMD commands** and **run C# code directly on your machine**.

These capabilities are extremely powerful and **potentially dangerous**.

Improper use may lead to:
- Data loss
- System modification
- Unintended file operations
- Full system-level execution behavior

👉 **Use with caution. Only run in trusted environments.**
